Первые шаги с git

  • Добавьте в репозиторий два файларазными коммитами:

      • Решение:

    1. cd ИМЯ ДИРЕКТОРИИ - Переходим в директорию
    2. echo 'I'm a student' > index.txt
    3. echo 'Я студент самоучка' > php.html
    4. git add index.txt
    5. git commit -m 'add index.txt'
    6. git add php.html
    7. git commit -m 'add php.html

Работа с git (Клонирование)

  • Клонируем репозиторий и изменяем файлы:

      • Решение:

    1. git clone https://github.com/anisimV/lesson.php.git
    2. cd ПЕРЕХОДИМ В НУЖНУЮ ДИРЕКТОРИЮ
    3. echo 'I'm a student' >> php.txt
    4. echo 'Git - контроль версий' > git.html
    5. git add .
    6. git config --global user.email "ВВЕДИТЕ ПОЧТУ"
    7. git config --global user.name "ВВЕДИТЕ ИМЯ"
    8. git commit -m 'fix php.txt and git.html'
    9. git push

Отмена изменений

  • Отменяем изменения двух файлов:

      • Решение:

    1. cd ПЕРЕХОДИМ В НУЖНУЮ ДИРЕКТОРИЮ
    2. git status -СМОТРИМ ИЗМЕНЕНИЯ
    3. git restore git.txt
    4. git restore php.html

Удаление/ изменения

  • Удаляем и изменяем файл:

      • Решение:

    1. cd ПЕРЕХОДИМ В НУЖНУЮ ДИРЕКТОРИЮ
    2. rm -f php.html.html

Отмена изменений

  • Отменяем все изменения, сделанные в рабочей директории и индексе

      • Решение:

    1. cd ПЕРЕХОДИМ В НУЖНУЮ ДИРЕКТОРИЮ
    2. git status
    3. git restore php.txt
    4. git restore --staged git.md git.html
    5. git restore git.html
    6. git clean -fd

Отмена коммитов

  • Удаляем два последних коммита

      • Решение:

Изменение последнего коммита

  • Изменяем последний коммит так, чтобы он содержал все три файла одновременно

      • Решение:

    1. cd ПЕРЕХОДИМ В НУЖНУЮ ДИРЕКТОРИЮ
    2. git add php.html
    3. git add git.md
    4. git commit --amend

Понимаем git

  • Заменяем содержимое тега p на «Я учусь коммитить»
  • Заменяем содержимое тега span на «Я студент»

      • Решение:

    1. cd ПЕРЕХОДИМ В НУЖНУЮ ДИРЕКТОРИЮ
    2. git switch refactoring
    3. git reset --hard HEAD~2
    4. git switch working-on-html
    5. # Редактируем и сохраняем файл
    6. vim php.html
    7. git add php.html
    8. git commit -m 'fix php.html'

Stash

  • Создаем файл php.md с любым содержимым
  • Добавляем в файл hexlet.txt строку: «Я студент, учу git» и делаем коммит
  • Восстановливаем из Stash сохраненные ранее изменения
  • Делаем еще один коммит

      • Решение:

    1. cd ПЕРЕХОДИМ В НУЖНУЮ ДИРЕКТОРИЮ
    2. git add .
    3. git stash
    4. echo 'Hexlet is awesome!' > php.md
    5. echo 'Я студент, учу git' >>git.txt
    6. git add git.txt
    7. git commit -m 'refactoring'
    8. git stash pop/li>
    9. git add .
    10. git commit -m 'changes from stash'
