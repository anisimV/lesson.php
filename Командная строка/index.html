Чтение файлов

  • Выведите на экран первые 5 строк файла /php/lesson/.bashrc прочитав их командой head.

Рещение:

"head -n 6 /php/lesson/.bashrc"

Grep

  • Грепните файл /php/lesson/.bashrc по строчке arr.

Рещение:

"grep arr /php/lesson/.bashrc"

Потоки

  • Прочитайте содержимое файла php и перенаправьте его в файл /res/result.

Рещение:

"cat php > /res/result"

Пайплайн

  • Посредством конвейера отсортируйте содержимое файла php в алфавитном порядке. Выведите на экран первые строки

Рещение:

"sort languages | head -3"

Переменные окружения

  • Глобально измените значение переменной на /lesson/php

Рещение:

"export MAIL=/lesson/php NEW_VAR=value"

Пользователи и группы

  • Dыведите на экран id пользователя ...

Рещение:

"id -u 'имя пользователя'"

Sudo

  • В директории php создайте файл index с текстом I'm a student!

Рещение:

"echo 'I'm a student!' | sudo tee php/index"

Права доступа

  • Измените права доступа к файлу ИМЯ ФАЙЛА

Рещение:

"chmod 511 ИМЯ ФАЙЛА"
